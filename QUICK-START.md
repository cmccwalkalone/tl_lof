# 🚀 快速诊断指南

## 立即开始

### 步骤 1: 访问诊断页面

在 Netlify 上访问：
```
你的域名/index-debug.html
```

例如：`https://my-app.netlify.app/index-debug.html`

### 步骤 2: 运行诊断

点击 **"重新加载数据"** 按钮

### 步骤 3: 查看结果

页面会显示：
- ✅ **统计数据** - 成功/失败计数
- 📊 **获取到的数据** - 如果成功会显示
- 📝 **详细日志** - 所有请求和错误信息

## 🎯 我需要你提供的信息

### 如果数据加载失败，请提供：

1. **页面截图** - 显示 `/index-debug.html` 的完整界面
2. **日志信息** - 复制日志面板中的所有内容
3. **控制台错误** - 按 F12，复制 Console 中的错误

### 如何复制日志：

1. 滚动到页面底部的日志面板
2. 点击每条日志可以展开查看详情
3. 复制所有日志条目
4. 粘贴给我

## 🔧 快速测试命令

### 测试单个 API

在浏览器中直接访问：

```javascript
// 测试基金净值 API
https://你的域名/api/fund?code=501312

// 测试腾讯价格 API
https://你的域名/api/tencent?code=501312

// 测试东财平层 API
https://你的域名/api/pingzhong?code=501312

// 测试基金信息 API
https://你的域名/api/fundinfo?code=501312
```

**预期结果**: 返回 JSON 数据

**404 错误** = Function 未部署或路由错误
